# valilik-talep-yonetim-sistemi
Valilikte stajyer olarak görev yaptığım sırada geliştirdiğim backend projem. --- A backend project I developed during my internship at the Governor’s Office.


İlk çalışabilir konsol demosu.

Özellikler:
- Kullanıcı doğrulama
- Rol bazlı menüler
- Talep oluşturma ve listeleme
- Teknisyene talep atama
- Talep durumunu işleme alma ve tamamlama
- String tabanlı kullanıcı, departman ve talep ID'leri

Bilinen sınırlamalar:
- Veriler bellekte tutulmaktadır
- Uygulama yeniden başladığında veriler silinir
- Demo hesaplarında düz metin şifre kullanılmaktadır
- CSV, bildirim, rapor ve yorum özellikleri henüz aktif değildir
